ChurnShield API v1

Integrate AI-powered churn prediction into your app. Works with Flipkart, Amazon, Zomato, or any business.

Quick Start

Get churn predictions for your customers in 3 simple steps:

1

Create Account

Sign up on ChurnShield and go to Settings.

2

Generate API Key

Create an API key from the dashboard.

3

Send Requests

Upload CSV or send JSON to get predictions.

Authentication

All API requests require an X-API-Key header.

# Include this header in every request X-API-Key: cs_your_api_key_here

Endpoints

POST /api/v1/predict
Upload a CSV or Excel file to get churn predictions for all customers.
# Python Example import requests url = "https://churnshield.ai/api/v1/predict" headers = {"X-API-Key": "cs_your_key"} files = {"file": open("customers.csv", "rb")} response = requests.post(url, headers=headers, files=files) print(response.json())
POST /api/v1/predict-json
Send customer data as JSON for real-time predictions. Perfect for integration with your backend.
# Python Example import requests url = "https://churnshield.ai/api/v1/predict-json" headers = { "X-API-Key": "cs_your_key", "Content-Type": "application/json" } data = { "customers": [ {"customer_id": "C001", "tenure": 12, "total_charges": 500}, {"customer_id": "C002", "tenure": 45, "total_charges": 3200}, {"customer_id": "C003", "tenure": 2, "total_charges": 80} ] } response = requests.post(url, headers=headers, json=data) print(response.json())
GET /api/v1/usage
Check your organization's API usage and remaining quota.
curl -H "X-API-Key: cs_your_key" https://churnshield.ai/api/v1/usage

Supported Data Formats

ChurnShield auto-detects your schema. No rigid format required.

E-Commerce (Flipkart, Amazon)

customer_id, order_date, amount, status, returns

Food Delivery (Zomato, Swiggy)

user_id, order_date, total_spend, frequency

Telecom (Jio, Airtel)

customer_id, tenure, monthly_charges, churn

Streaming (Netflix, Hotstar)

account_id, subscription_months, total_charges
